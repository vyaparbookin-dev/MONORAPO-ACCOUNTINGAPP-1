# Vyapar Accounting - Marketing Website

This is the official marketing and landing page for the Vyapar Accounting application. It is built with React and Vite.

## Project Structure

This project is part of a monorepo. The shared code, components, and services are located in the `packages/` directory at the root of the repository.

## Development

To run this website locally, navigate to the root of the monorepo and run:

```bash
npm run dev
```

This will start the Vite development server for all applications in the monorepo, including this one.
